projects

  • The Public Note – Smart Note Space for Public Sharing

    The Public Note is web application designed to allow users to create, store, and share notes publicly through unique, instantly generated links. The platform focuses on simplicity, speed, and anonymity, making it ideal for scenarios such as seminars, public announcements, quick information sharing, or anonymous tips.


    Built as a solo project, the application follows a modern full-stack architecture using React.js (Vite) for the frontend and Node.js with Express.js for the backend, while MySQL is used for secure and structured data storage. The frontend delivers a clean and responsive user experience, and API communication is handled efficiently using Axios and React Router.


    The system allows users to create notes without mandatory authentication, automatically generating a shareable link that provides instant access to the content. Notes are securely stored in the database, ensuring reliability and scalability. The design prioritizes usability and fast access while maintaining data integrity. Future enhancements include user authentication, private note locking, the ability to edit notes after saving, and support for personal dashboards—making the platform adaptable for both public and private use cases.


    Tech Stack: React.js, Vite, Node.js, Express.js, MySQL, Axios, React Router, Custom CSS
